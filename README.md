There are two main folders, Python and QGIS. 
- The Python part consists of a main folder and two sub-folders.
  - The main folder contains the scripts written.
  - In the sub-folder ‘provided_csv’ are the csv files stored that are used as the defaults in the scripts.
  - In the sub-folder ‘gpkgs’ the used GeoPackages are stored. 
- The QGIS part consists of a main folder with only one sub-folder.
  - In the main folder the project is stored.
  - The subfolder contains all the GeoPackages.
 

Data Sources:

CSV files provided by M. R. Groenhuijzen and P. Verhagen. 
  - The csv files are part of the ongoing project “Simulating Globalisation in the Roman Empire” and contain data from ARCHIS. 

Palaeogeography shapefile provided by M. R. Groenhuijzen.
  - The palaeogeography file is part of the ongoing project “Constructing the Limes”, and the first part (Rivierengebied) was constructed during his PHD project.
    - Groenhuijzen, M. (2018). Palaeogeographic analysis of the dutch part of the Roman Limes and its hinterland [Doctoral dissertation, Vrije Universiteit Amsterdam].

The provincial boundaries of the Netherlands were created by Centraal Bureau voor de Statistiek (CBS), and was found on Publieke Dienstverlener Op de Kaart (PDOK).
  - https://www.pdok.nl/introductie/-/article/cbs-gebiedsindelingen
